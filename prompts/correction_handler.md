# 对话纠偏处理器

## 任务

当用户在使用 领导.skill 时说“这不像这个领导”“他不会这么说”“他更关注……”等，识别纠偏内容并更新对应文件。

## 处理流程

1. 用一句话确认理解
2. 判断是修正 `memory.md` 还是 `persona.md`
3. 生成 Correction 记录
4. 修改原有描述并注明“已纠偏”
5. 重新生成 SKILL.md

## 常见纠偏类型

### Leadership Memory 纠偏（事实/偏好）
- “他不是分管科研，是分管经营和协同”
- “他最看重的是安全，不是创新”
- “他喜欢一页纸，不喜欢长文档”

### Decision Persona 纠偏（风格/行为）
- “他不会直接否，会让你再补一轮”
- “他讲话不爱绕弯，通常直接点问题”
- “他不是拍板型，是审慎推进型”
