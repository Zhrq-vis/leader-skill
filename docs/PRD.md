# 领导.skill — 产品需求文档（PRD）

## 产品定位

领导.skill 是一个运行在 Claude Code 上的 meta-skill。
它将某位领导的讲话、批示、纪要、材料偏好和真实决策案例，蒸馏为一个可复用的“领导预审与决策模拟器”。

其核心用途不是“角色扮演”，而是帮助用户在正式汇报前完成：
- 方案预审
- 口径改写
- 会场追问模拟
- 风险边界校验
- 批示意见草拟

## 两层架构

| 层 | 名称 | 职责 |
|----|------|------|
| Part A | Leadership Memory | 存储岗位背景、常抓重点、风险底线、偏好案例、典型批示 |
| Part B | Decision Persona | 决定说话方式、追问方式、拍板口径、风险偏好、会场行为 |

## 用户旅程

1. 触发 `/create-leader`
2. 输入领导代号、岗位背景、风格画像
3. 导入讲话、纪要、批示或口述材料
4. 自动提取 Leadership Memory + Decision Persona
5. 预览摘要并确认
6. 生成 `/{slug}` 可调用的领导 Skill
7. 后续持续追加材料、纠偏、回滚版本

## 核心价值

- 把“领导经验”从模糊印象变成结构化规则
- 把“汇报翻车点”前移到会前演练阶段
- 把“领导风格差异”沉淀为可复用组织资产

## 典型输出

- `这个方案领导会不会批？`
- `这页 PPT 领导会怎么改？`
- `这件事领导会追问什么？`
- `帮我生成一段领导批示意见`
